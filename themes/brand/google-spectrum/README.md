# Google Spectrum

Google Spectrum is an unofficial Codex Desktop visual theme by VIIS Labs. It combines an original software-ecosystem illustration with semantic Google color roles across the native Codex sidebar, suggestions, composer, project controls, dialogs, messages, and primary actions.

The artwork is displayed without a full-window mask. The pack contains no Google logo or wordmark, and the compatibility runtime does not place a floating logo over the Codex workspace.

## Local validation and installation

From the repository root:

```sh
node packages/theme-cli/bin/get-codex-theme.mjs validate themes/brand/google-spectrum --strict-assets
node packages/theme-cli/bin/get-codex-theme.mjs install themes/brand/google-spectrum
```

After review, close Codex and use the installed macOS or Windows start script to activate the unofficial CDP visual layer. Use the matching restore script to return to the stock appearance. Validation itself is read-only and does not connect to Codex.
